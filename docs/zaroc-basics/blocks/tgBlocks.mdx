# tgBlocks

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Telegram Blocks help us to display information on Telegram

Telegram Blocks have a Native Integration with Dialogflow, and there are others that uses external Payloads

### tgText

**Is used to show Texts**

> Multiple texts are stacked as paragraphs.

![Download this document](./../../../static/img/blocks/tg/text.jpg)

<Tabs>
<TabItem value="php" label="PHP">

```php
$tTextArray = array("Text1","Text2");

tgText($tTextArray);
```

</TabItem>

### tgReply

**Is used to show Replies/Chips**

> - Chips are stacked horizontaly up to 7.
> - Chips functions as a common text input.

![Download this document](./../../../static/img/blocks/tg/reply.jpg)

<Tabs>
<TabItem value="php" label="PHP">

```php
$rTitleArray = array("Title");
$rTextArray = array("option1","option2");

tgReplies($rTitleArray, $rTextArray);
```

</TabItem>

### tgCard

**Is used to show Cards**

> - Cards are Stacked vertically.
> - Button Cards support Links and Text.

![Download this document](./../../../static/img/blocks/tg/card.jpg)

<Tabs>
<TabItem value="php" label="PHP">

```php
$cImageArray = array("https://ramenparados.com/wp-content/uploads/2018/06/Barakamon-destacado-1000x600.jpg");
$cTitleArray = array("CardTitle1");
$cSubtitleArray = array("Card subtitle1");
$cButtonTextArray = array(array("button1","button2"));
$cButtonCustomArray = array(array("button1","button2"));

tgCard($cImageArray,$cTitleArray,$cSubtitleArray,$cButtonTextArray,$cButtonCustomArray);
```

</TabItem>
