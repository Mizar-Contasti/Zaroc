### Kael, Chatbot that Sells Chatbots. Check out [**HERE**](https://mizar-contasti.github.io/Kael/)

> - This Chatbot supports English and Spanish, saves Engages and Session on Db.
> - Leads the user into a Tutorial to show benefits of a chatbot for user specific business case.
> - Kael is able to send Emails, and follow the user if theres an Error or missunderstanding.

### Alicia, Hospital Care and Administration Chatbot. Check out [**HERE**](https://mizar-contasti.github.io/Alicia/)

> - Alicia helps the user to signUp, create an Account, a Medical History and Profile.
> - User can have multiple roles and features (Pacient, Doctor, Admin).
> - Alicia helps Pacients to Schedule an Appointment with their preferred Doctors and Specialists.
> - Pacients are able to see the entire profile of the Doctor.
> - Doctors are able to see all the Appointments that they have, see the medical story and profile of the pacient.
> - Alicia stores information (Payments, Bookings, User Data in General) in DB. Send Emails with secretCodes, confirmation of Appointments.
> - Alicia offer services to recoverPassword, closeSessions, change userData.

### Thomas, Banking Chatbot. Check out [**HERE**](https://misaelbastidas.github.io/Thomas/)

> - Thomas is a Chatbot that helps the user to get Information about the Bank.
> - Thomas helps us the user to Create an Account at the bank, also provides a menu where the user can see all the movements, transactions, banking profile and more.
> - Thomas Register all the transactions, helps the user to calculate credits, obtain credits and credit cards.
